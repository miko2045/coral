# Contributing to Coral Portal

Thank you for considering contributing to the Coral Portal project! We welcome contributions from everyone. Here are some guidelines to help you get started:

## Ways to Contribute
1. **Reporting Issues**: If you find a bug, please report it in the Issues section of the repository. Include a clear description, steps to reproduce, and any relevant screenshots.
2. **Requesting Features**: Have a great idea for a feature? Create a new issue and describe the feature in detail.
3. **Submitting Pull Requests**: If you'd like to contribute code, please do the following:
   - Fork the repository.
   - Create a new branch for your feature or bugfix.
   - Write clear, concise commit messages.
   - Ensure your code follows the project's coding style and includes tests, if applicable.
   - Submit a pull request with a description of your changes.

## Code of Conduct
Please adhere to the [Code of Conduct](link-to-code-of-conduct) while contributing. 

## Sign Your Work
Please sign your commits to certify that you wrote or modified the code. For more information on this,
see the [Developer Certificate of Origin](https://developercertificate.org/).
